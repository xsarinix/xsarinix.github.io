Latitude & Weather

Conventional wisdom states that as you get closer to the equator, the weather gets warmer. But how do we really know? Data's true power lies in its ability to answer questions definitively. This project uses the citipy Python library and data pulled from the OpenWeatherMap API to visualize the weather of over 500 cities across the globe to answer this question.

I generated 1500 random latitudes and longitudes with numpy. Using citipy, these coordinates were matched to the nearest city, with duplicates being dropped, resulting in 618 unique cities. The most recent measurements of cloudiness, humidity, wind speed, and high temperature were then pulled by city and country using the OpenWeatherMap API, skipping unfound cities, for a final count of 536 cities.

The figures to the right are the results of comparing latitude to the four metrics above using matplotlib. Detailed analysis for each plot can be accessed by clicking on the figure or from the Plots dropdown in the navigation bar. An analysis of all graphs can be found under the comparisons page, and the data used for the analysis can be found under the data tab.
